## 3. Профилирование: cProfile

### Что анализируется:
- Время выполнения каждой функции
- Количество вызовов функций
- Накопленное время (cumulative time)
- Собственное время функций (tottime)

### Использование результатов:
```bash
# Просмотр профиля
python view_profile.py

# Или интерактивно
python -m pstats analysis_results/profile.prof
```

### Метрики производительности:
- Общее время выполнения
- Самые медленные функции
- Функции с наибольшим количеством вызовов
- Узкие места (bottlenecks)
